# Configuration des fichiers d'envirronement

## `auth-bdd.env`

### Structure

```env
MARIADB_ROOT_PASSWORD=
MARIADB_HOST=
MARIADB_DATABASE=
MARIADB_USER=
MARIADB_PASSWORD=
```

### Exemple

```env
MARIADB_ROOT_PASSWORD=oui
MARIADB_HOST=Reunionou_api_db
MARIADB_DATABASE=reunionou
MARIADB_USER=api_client
MARIADB_PASSWORD=non
```
